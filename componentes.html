Tablas

Documentación y ejemplos para el estilo opcional de tablas

Debido al uso generalizado de tablas en widgets de terceros, como calendarios y selectores de fechas, hemos diseñado nuestras tablas para que sean habilitadas. Simplemente agregue la clase base .table a cualquier table, luego amplíe con estilos personalizados o nuestras diversas clases de modificadores incluidas.

# Name Lastname Country
