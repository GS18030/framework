Contenedores



Todo maquetacion hecho con el framework debera tener como contendor padre un container.

Zeus cuenta con dos tipos de contenedores.

  • container, este contendor posee un maximo de 1300px de ancho
  • container-wrap, este contenedro tiene un ancho del 100%.

Estructura de un contenedor

En princio la estructura de un contenedor es muy simple, primero se estable un container como elemento padre que almacenara todos los elementos html, luego dentro de este se coloca un subcontenedor con las clase (puede ser tantos como quiera) y dentro de esto podra establecer las columnas que ocupa los elementos html.

Como se puede ver a continuacion...

            
                <div class="container">
                    <div class="row">
                        <div class="col-3">
                            Contenedor 1
                        </div>
                        <div class="col-4">
                            Contenedor 2
                        </div>
                        <div class="col-3">
                            Contenedor 3
                        </div>

                    </div>
                </div>

Zeus es un framework hecho para facilitar el responsive, Zeus utiliza los siguientos medias query para ello. 

            
                @media (max-width: 480px){ 
                    codigo...
                }

                @media (min-width: 481px ) and (max-width: 720px) {
                    codigo...
                }

                @media (min-width: 721px) and (max-width: 960px){
                    codigo...
                }

                @media (min-width: 961px) and (max-width: 1200px){
                    codigo...
                }

                @media (min-width: 1201px){
                    codigo...
                }

Filas

Un container puede tener cuantas filas se necesite, las filas son contenedores importantes ya que permiten definer las columnas que los elementos dentro de estas tendra, lo que mejora el desarrollo UI.

Tenemos la clase row para fila.

Columnas

Las columnas es una de las cosas mas importantes en Zeus ya que permiten determinar cuanto espacio ocupara un elemento en una fila y facilita el responsive del sitio web.

Cabe mencionar que en nuestro framework hay un maximo de 10 columnas, es decir un elemento solo podra ocupar un maximo de 10 columnas.

Margenes

Utilizar margenes es algo muy sencillo si se utiliza las clases de Zeus, como lo veremos... 

              
                  <h1 class="m3">
                      Margenes
                  </h1>

Tambien podemos poner margenes automaticos en los ejes x o y 

              
                  <h1 class="mx-auto">
                      Margenes automaticos en el eje x
                  </h1>

Padding

Algo importante a observar es que se puede poner paddins en los contenedores y contenidos sin preocuparse a que este afecte el tamaño del elemento, la manera de poner un padding es como se muestra a continuacion. 

              
                  <h1 class="p3">
                      Relleno
                  </h1>

Clases Extras

Zeus ademas probee de unas clases extras para facilitar la maquetacion de un sitio web

Anchura y Altura

Podemos definir el ancho de un elemento dentro los valores 25%,50%, 75%, 100%. Como se muestra a continuacion 

              
                  <div class="w-100">
                      Ancho de la caja
                  </div>

Igualmente podemos definir la altura de un elemento dentro los valores 25%,50%, 75%, 100%. Como se muestra a continuacion 

              
                  <div class="h-100">
                      Altura de la caja
                  </div>
